# 🏦 Simple Bank Client Management System (C++)

A **console-based banking app** written in C++ to manage clients and transactions.  
It stores data in a text file and provides a menu-driven interface.

---

## 📌 Features

- **Add clients** (no duplicate account numbers)
- **Show all clients** in a table
- **Find client** by account number
- **Update client** (keep old values with Enter)
- **Delete client** (soft delete with confirmation)
- **Deposit / Withdraw** money (with validation)
- **Show total balances**
- **Sort clients** by account number automatically

---

## 🚀 How to Run

1. **Clone or download** the repository
2. Open `C++ Bank Whith Extension pro 2.cpp` in Visual Studio or any C++ IDE
3. Compile and run (Windows only, uses `system("cls")`)

---

## 📁 File Format

Data is stored in `ZDFile.txt` using `#//#` as separator:
AccountNumber#//#PinCode#//#Name#//#Phone#//#Balance
